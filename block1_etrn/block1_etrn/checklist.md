# Чек-лист тестирования ЭТрН

## 1. Создание документа

- [ ] Успешное создание ЭТрН с заполнением всех обязательных полей
- [ ] Создание ЭТрН с минимальным набором полей
- [ ] Невозможность создания ЭТрН без обязательных полей
- [ ] Проверка формата полей (дата, время, ИНН, КПП)

## 2. Валидация полей (по XSD-схеме)

- [ ] Проверка всех обязательных реквизитов
- [ ] Проверка форматов (числовые, строковые, даты)
- [ ] Проверка допустимых значений (справочники)

## 3. Статусная модель

- [ ] Черновик → Заполнена
- [ ] Заполнена → Отправлена
- [ ] Отправлена → Подписана грузоотправителем
- [ ] Подписана грузоотправителем → Подписана перевозчиком
- [ ] Подписана перевозчиком → Доставлена
- [ ] Отказ в подписании → Статус ошибки

## 4. Отправка и подписание

- [ ] Успешная отправка контрагенту
- [ ] Корректное отображение приглашения на подписание
- [ ] Подписание ЭП (электронной подписью)
- [ ] Отказ от подписания с комментарием

## 5. Обработка ошибок

- [ ] Ошибка при отправке без ЭП
- [ ] Ошибка при превышении максимального веса/объёма
- [ ] Ошибка при несовпадении реквизитов
- [ ] Таймаут при отправке – повторная попытка
