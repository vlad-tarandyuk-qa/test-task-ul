# Граничные условия для тестирования формы создания заявки на перевозку

## Вес груза (кг, максимум 20 000)
- **0** → ошибка «Значение должно быть больше 0»
- **1** → допустимо (минимальный положительный вес)
- **20 000** → допустимо (максимум)
- **20 001** → ошибка «Превышен максимальный вес»
- **дробные значения** (10.5) – допустимо, если бизнес-логика разрешает; иначе ошибка формата

## Объём груза (м³, максимум 82)
- **0** → ошибка
- **0.1** → допустимо
- **82** → допустимо (максимум)
- **82.1** → ошибка превышения

## Комбинации вес + объём (критично, но в требованиях не указано – требуют уточнения)
- **Вес = 20 000 кг, объём = 0.1 м³** → невозможная плотность (нужно предупреждение)
- **Вес = 1 кг, объём = 82 м³** → пустой фургон (логическое предупреждение)

## Дата погрузки
- **Вчера** → недоступна в датапикере
- **Сегодня** → доступна
- **Завтра** → доступна
- **Будущая дата (через 6 месяцев)** → доступна
- **29 февраля в невисокосный год** (например, 29.02.2027) → дата должна блокироваться или приводиться к 28.02 / 01.03

## Автокомплит городов
- **Ввод 1–2 символов** → появляется список вариантов
- **Ввод с опечаткой** → список похожих или пусто
- **Несуществующий город** → «Город не найден»
- **Город с буквой «ё»** («Орёл») → находится
- **Два города с одинаковым названием** → отображается регион или уточнение

## API
- **Маршрут не найден** → ошибка 404, понятное сообщение
- **Таймаут (>30 сек)** → сообщение «Сервис временно недоступен»
- **Ответ с невалидным JSON** → ошибка обработки, не падение
- **Пустой ответ (200, но нет поля price)** → сообщение «Ошибка расчёта»
- **Протухший токен** → 401, перенаправление на авторизацию

## Кнопка «Рассчитать»
- **Двойной клик** → отправляется только один запрос, кнопка блокируется на время запроса
- **Клик при пустых обязательных полях** → запрос не отправляется, подсветка полей

## Специальные символы
- **Ввод текста в числовое поле** («abc») → поле не принимает, либо ошибка
- **Вставка из буфера (Ctrl+V)** → корректная обработка
- **Спецсимволы в числовых полях** (`<`, `>`, `&`, `"`, `;`) → игнорируются или ошибка
- **Пробелы в числах** → обрезаются или ошибка
